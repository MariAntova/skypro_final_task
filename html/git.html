GitHub

Основные

  • git init - создать новый проект в текущей директории.
  • git clone - клонировать репозиторий в текущую директорию
  • git add text.txt - добавить в индекс указанный файл
  • git commit -m "Name of commit" - зафиксировать в коммите проиндексированные изменения (закоммитить), добавить сообщение
  • git remote -v - показать список удалённых репозиториев, связанных с локальным
  • git push - отправить в удалённый репозиторий
  • git status - показать состояние репозитория (отслеживаемые, изменённые, новые файлы)

Ветки

  • git branch - показать список веток
  • git branch -v - показать список веток и последний коммит в каждой
  • git branch new_branch - создать новую ветку с указанным именем на текущем коммите
  • git checkout new_branch - перейти в указанную ветку
